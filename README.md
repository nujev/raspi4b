# raspi4b

Repositorio de **dotfiles, scripts y configuraciones** para una **Raspberry Pi 4B+**.

La idea es usar este repo como **fuente de verdad** para:

- reinstalar desde cero;
- recuperar la configuración tras una incidencia;
- reaplicar la base del sistema de forma consistente.

## Convención de trabajo

Cada avance útil debe **quedar versionado en Git y publicarse en remoto** en cuanto esté listo, para no perder cambios y mantener este repositorio siempre sincronizado con el estado real de la máquina.
